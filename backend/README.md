# Flourish Backend

API server (Express + Postgres) that implements the base44-compatible contract for the Flourish frontend.

**For prerequisites, setup, running, and verification, see the [root README](../README.md).**

- Schema: `schema.sql`  
- Seed: `seed.sql` (run with `npm run db:seed`)  
- Env: copy `.env.example` to `.env` and set `DATABASE_URL`, `PORT`
