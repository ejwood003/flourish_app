# Flourish Frontend

React + Vite app for Flourish (mood, journal, baby tracking, affirmations, support).

**For prerequisites, setup, running, and verification, see the [root README](../README.md).**

- Set `VITE_BASE44_APP_BASE_URL=http://localhost:3000` and `VITE_BASE44_APP_ID=your_app_id` in `.env` or `.env.local`, then run `npm run dev`.
